# ohana.css

## Install

```
$ git clone git@github.com:baooab/ohana.css.git
$ cd ohana.css
$ npm install
```

## Development mode

### Watch

```
$ npm run wacth
```

or build directly.

### build

```
$ npm run build
```

The final generate's `ohana.css` locate at `dist/css/hana.css`, feel free to use it :)
