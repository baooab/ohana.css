# ohana.css

## Install

```
$ git clone git@github.com:baooab/ohana.css.git
$ cd ohana.css
$ npm install
```

## Development mode

### watch

```
$ npm run watch
```

or build directly.

### build

```
$ npm run build
```

The resulting file `ohana.css` locates at `dist/css/ohana.css`, feel free to use it :)
